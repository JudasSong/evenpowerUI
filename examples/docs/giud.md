# EvenpowerUI

----

`Evenpower-UI` 是一款基于 `Vue` ，在项目中使用```element-ui``` 过程中积累一些编写组件经验，利用闲暇之余（**吃饱了撑着**） 开发出来用于学习交流的前端 UI 组件库

## 由来

- 公司项目需要，毕竟每个公司都应该有自己的组件库
- 更好的学习 `element` 
- 与其不一样的是，项目的样式采用的是less。
- 虽然大部分是致敬的，但也加入一些自己的组件。
- 一些项目用到的导航菜单自适应浏览器窗口大小


## 问题

在使用 `Evenpower-UI` 时遇到问题，或者有好的建议，欢迎提出来，以便更好服务于大众
